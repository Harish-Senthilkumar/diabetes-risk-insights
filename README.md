# Diabetes Risk Insights

## Overview
This project is supposed to analyze the CDC Diabetes Health Indicators dataset to see which health, lifestyle, and demographic factors are most strongly correlted with diabetes risk, with the goal of understanding what combos of indicators are the reason/likelihood of having diabetes!

## Data Source and Proper Accreditation
[CDC Diabetes Health Indicators](https://archive.ics.uci.edu/dataset/891/cdc+diabetes+health+indicators)
via the UCI Machine Learning Repository.

## Project Structure
This project shows and follows the full data science lifecycle (including Data Analyst)
  1. Data Ingestion
  2. Data Cleaning and Preparation
  3. Explaratory Data Analysis (EDA)
  4. Machine Learning Modeling
  5. Storytelling and Insights
